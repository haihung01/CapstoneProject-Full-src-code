# TripTix - Bus Ticket Booking System

#### TripTix is an online bus ticket booking application for users. The application was developed by a group of young people with the desire to bring users a simple, convenient and safe online bus ticket booking experience.

## 1.Major features
- As a TripTix system administrator, I want to manage trips so I can see how many trips are running in the system. I want route management so I can control the trips that work for my company. I want to manage passenger cars so I can know how many passenger cars are in the system. I want to manage stops so I can see how many stops or stops there are so I can create a route.
